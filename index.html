Graph Info

How we came up with the graph: We started by coming up with zeros that are multiples of 250, in order to make the number we use to multiply by pleasing to the eye. Then we saw what the y-intercept was for the graph, and did 250⁄(y-intercept). In our case, we got 8⁄5. Then we gave x a coefficient in order to change the graph.

  • Factored Form: 8⁄5(x-4)(x-5)(0.5x-2.5)(x-3.125)(2x-1)2(x2+1)

    Standard Form: 3.2x8 - 58x7 + 406.8x6 - 1389.7x5 + 2460.6x4 - 2574.2x3 + 2307x2 - 1242.5x + 250

  • As x → ∞, f(x) → ∞

    As x → -∞, f(x) → ∞

  • Domain: (-∞, ∞)

    Range: (-162.85, ∞)

  • Minima: (0.5, 0), (3.54, -162.85), (5, 0)

    Maxima: (2.03, 729.92), (4.47, 187.91)
